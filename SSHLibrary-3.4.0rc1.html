SSHLibrary 3.4.0 Release Candidate 1

SSHLibrary is a Robot Framework test library for SSH and SFTP. SSHLibrary 3.4.0 Release Candidate 1 is a new release with several enhancements and bug fixes. All issues targeted for SSHLibrary v3.4.0 can be found from the issue tracker.

If you have pip installed, just run

pip install --pre --upgrade robotframework-sshlibrary

to install the latest release or use

pip install robotframework-sshlibrary==3.4.0rc1

to install exactly this version. Alternatively you can download the source distribution from PyPI and install it manually.

SSHLibrary 3.4.0rc1 was released on Tuesday July 30, 2019.
